# Node Red

For more complex smart home automation. It can connect to devices on the wifi as well. It can be used for automation on it own. However it can also be triggered via home Assistant.

# Configuration

Install Node-RED on Home Assistant via HACS. 

- Make a Node-RED User in Home Assist
- Configure the Node-RED Integration by pointing to the node-RED service


Adding Home Assistant pallet in Node Read

- Connect Node Red to Home Assistant
    - Install Home Assistant Websockets palette
      - Hamburger Menu -> Manage Palette
      - Install tab
      - Search For node-red-contrib-home-assistant-websocket
      - click install
